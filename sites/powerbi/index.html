DAX Functions

DAX (Data Analysis Expressions) is a formula expression language used in Power BI, Excel Power Pivot, and Analysis Services to create custom calculations, measures, and calculated columns.

Aggregates

FunctionDescription
Average Functions
⚡AVERAGE(column) → DecimalReturns the arithmetic mean of a numeric column.
Includes: Numeric values.
Ignores: Text and blanks.
⚡AVERAGEA(column) → DecimalReturns average treating non-numeric values as 0, logical as TRUE = 1, and FALSE = 0.
Includes: Text, logical, and numeric values.
Ignores: Blanks.
⚡AVERAGEX(table, expression) → DecimalReturns average of an expression evaluated over a table.
Includes: Expression results per row.
Ignores: Blanks from the expression.
Count Functions
⚡COUNT(column) → IntegerCounts numeric values in a column.
Includes: Numeric values.
Ignores: Text, logical, and blanks.
⚡COUNTA(column) → IntegerCounts non-blank values.
Includes: All non-blank values.
Ignores: Blanks.
⚡COUNTAX(table, expression) → IntegerCounts rows where expression evaluates to a non-blank value.
Includes: Non-blank expression results.
Ignores: Blank expression results.
⚡COUNTX(table, expression) → IntegerCounts rows where the expression evaluates to a numeric value.
Includes: Numeric values from the expression.
Ignores: Non-numeric and blank values.
COUNTBLANK(column) → IntegerCounts blank values in a column.
Includes: Blank values.
Ignores: Non-blank values.
⚡COUNTROWS(table) → IntegerCounts the number of rows in a table.
Includes: All rows.
Ignores: Blank values are counted but blank rows are included.
⚡DISTINCTCOUNT(column) → IntegerCounts distinct non-blank values in a column.
Includes: Unique non-blank values.
Ignores: Duplicates and blanks.
DISTINCTCOUNTNOBLANK(column) → IntegerCounts distinct values, ignoring blanks.
Includes: Unique values.
Ignores: Blank values.
Max/Min Functions
⚡MAX(column) → NumericReturns the largest value in a column.
Includes: Numeric values.
Ignores: Non-numeric and blank values.
⚡MAXA(column) → NumericReturns the largest value in a column, treating non-numeric as 0 and logical as TRUE = 1, FALSE = 0.
Includes: Text, logical, and numeric values.
Ignores: Blanks.
⚡MAXX(table, expression) → NumericReturns the largest value of an expression evaluated over a table.
Includes: Expression results.
Ignores: Blank or non-numeric results.
⚡MIN(column) → NumericReturns the smallest value in a column.
Includes: Numeric values.
Ignores: Non-numeric and blank values.
MINA(column) → NumericReturns the smallest value in a column, treating non-numeric as 0 and logical as TRUE = 1, FALSE = 0.
Includes: Text, logical, and numeric values.
Ignores: Blanks.
⚡MINX(table, expression) → NumericReturns the smallest value of an expression evaluated over a table.
Includes: Expression results.
Ignores: Blank or non-numeric results.
Product/Sum Functions
⚡PRODUCT(column) → NumericReturns the product of all numeric values in a column.
Includes: Numeric values.
Ignores: Non-numeric and blank values.
PRODUCTX(table, expression) → NumericReturns the product of an expression evaluated over a table.
Includes: Expression results.
Ignores: Blank or non-numeric results.
⚡SUM(column) → NumericReturns the sum of all numeric values in a column.
Includes: Numeric values.
Ignores: Non-numeric, logical, and blank values.
⚡SUMX(table, expression) → NumericReturns the sum of an expression evaluated over a table.
Includes: Expression results.
Ignores: Blank or non-numeric results.
