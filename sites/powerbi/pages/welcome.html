    Extensive workflow

    Connect

    Source types - Files, Databases, Power Platform, Online Services, Others, Enter data
    Connection - Data Source Settings
    • Location: Path / Server / URL
    • Object: Filename / Database / Semantic Model / API
    • Query: SQL statement / Stored Procedures / Table Valued Functions
    • Credentials: UName Pwd / Token / Keys / Signin / Anynomous
    • Privacy Level: Public / Organizational / Private
    Connection Mode - Import / Direct Query / Live Connection
    Object Selection - Sheets / Table, Table / Views / Cube-Dimensions, Semantic Model / Dataflow

    Transform

    Building Blocks - Table, List, Functions, Parameters
    Explore - Column Profile, Column Distribution, Column Quality
    Applied Steps -
    • Manage Parameter: RangeStart/RangeEnd, FilePath/FolderPath, ServerName/DatabaseName, CutoffData/AsOfDate, Environment
    • Query: Enable Load, Duplicate, Reference, Convert to function/list
    • Table Transformations: Append, Merge, Group by, Transpose, 1st Row as Headers, Reverse Rows, Count Rows
    • Column Transformations: Data type, Rename, Pivot / Unpivot, Replace, Fill, Convert to List, Derived/Custom column, Conditional column, Duplicate column, Extract / Split / Merge, Format, Parse, Select / Remove, Transform, Lookup, Expand, Statistics, Rounding
    • Scripts: R, Python
    • Diagnostics: Start, Stop

    Data Model

    Building Blocks - Table / Calculated Table, Relationship, Direction, Storage Mode, Calculated Column, Data Type, Format, Sort By, Calculation Group, Partition, Perspectives, Hierarchy, Measures, User Defined Functions
    Storage Mode - Mode: Import / Direct Query / Dual, Conversions, Irreversible conversions
    Relationship - Type: One-to-One / One-to-Many / Many-to-Many, Direction: Single / Bi-directional
